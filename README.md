# Minecraft 版本列表
一个**现代化、高颜值、响应式**的 Minecraft 版本查询工具，支持版本筛选、搜索、分页、详情查看。

## ✨ 功能特性
- 🎨 **现代 UI 设计**：玻璃态渐变 + 明暗主题自动适配
- 🔍 **全局搜索**：支持版本名称、描述关键词搜索
- 🏷️ **标签筛选**：正式版、快照版、Beta、Alpha 等一键筛选
- 📄 **分页展示**：支持自定义每页条数，上下双分页控件
- 🔗 **相关链接**：弹窗查看版本下载/官方链接
- 📝 **详情查看**：弹窗展示完整版本描述
- 🧩 **URL 状态同步**：刷新页面保留筛选/搜索/页码
- 📱 **完美响应式**：适配电脑、平板、手机等设备
- 🚫 **无冗余数据**：已自动过滤“跳过”等无效条目

## 📁 文件结构
```
├── index.html       # 主页面结构
├── style.css        # 完整样式（主题+动画+响应式）
├── script.js        # 核心逻辑（筛选+分页+弹窗）
└── versions.json    # 版本数据文件（必须自行提供）
```

## 🚀 快速使用
1. **下载所有文件**，放到同一目录
2. **准备版本数据**：在目录中放入 `versions.json` 数据文件
3. **打开 `index.html`**，即可直接使用
4. 由于浏览器安全限制，无法直接访问本地json文件，请使用`python -m http.server`等工具启动本地服务器。

## 🎯 使用说明
### 1. 搜索版本
在顶部搜索框输入关键词，按回车或点击「搜索」即可。

### 2. 标签筛选
点击标签进行筛选，**同组标签互斥**（如正式版与快照版不能同时选）。

### 3. 分页控制
- 上下均有分页栏，操作同步
- 支持切换每页显示数量（5~500条）
- 第一页自动禁用上一页，最后一页自动禁用下一页

### 4. 查看详情
- **详情**：查看完整版本描述
- **查看**：打开弹窗获取相关下载/官方链接

## 🎨 界面预览
- 顶部渐变玻璃态头部 + 搜索框
- 圆角标签筛选栏，支持激活态
- 精致表格样式，悬停效果
- 现代弹窗设计，统一主题风格
- 明暗主题自动跟随系统设置

## 🔧 自定义配置
### 修改主题色
打开 `style.css`，修改 `:root` 中的 CSS 变量：
```css
:root {
  --primary: #6366f1;      /* 主色调 */
  --primary-dark: #4f46e5; /* 深色主色调 */
  /* ... */
}
```

### 修改分页默认值
打开 `script.js`，修改：
```javascript
let pageSize = 50;    // 默认每页条数
```

## 📌 版本数据格式（versions.json）
数据必须为数组，每个版本格式：
```json
[
  {
    "name": "1.21",
    "date": "2024-05-10",
    "description": "正式版更新内容...",
    "tags": ["正式版"],
    "downloads": {
      "client": "https://piston-data.mojang.com/v1/objects/...",
      "json": "https://piston-data.mojang.com/v1/objects/...",
      "server": "https://piston-data.mojang.com/v1/objects/...",
      "wiki": "https://zh.minecraft.wiki/w/...",
      "minecraft": "https://www.minecraft.net/zh-hans/article/minecraft-java-edition..."
    }
  }
]
```
